汎用データベース接続パッケージ

汎用データベース接続パッケージ。


実装方法は、Clientクラスを参照。

プロパティファイル( set.properties )を作成し、以下の項目を設 定すること。
dbTypeで、MySqlConクラス、PostgreSqlConクラス、SqlServerConク ラスを指定することで、RDBMSの切替が出来る。


#sys.db.dbType = com.tdtsh.marusen.database.PostgreSqlCon #sys.db.dbType = com.tdtsh.marusen.database.SqlServerCon sys.db.dbType = com.tdtsh.marusen.database.MySqlCon sys.db.hostname = localhost
sys.db.dbname = mailmagazine
sys.db.user = slode
sys.db.pass = mmuser
sys.db.encoding = utf-8
